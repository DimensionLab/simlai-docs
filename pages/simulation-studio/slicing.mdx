# Slicing with planes

Siml.ai Simulation Studio lets you slice through the simulation outputs in various ways. Slicing functionality can be toggled in the
right menu by clicking on any axis name (Fig. 1).

<Image
    src="/img/simlai-simulation-studio-slice-1.png"
    alt="Slice through the simulation domain"
    width={1280}
    height={720}
    style={{ margin: "auto" }}
/>
<figcaption style={{ textAlign: "center", fontSize: "0.7rem", marginBottom: "1rem" }}>
    Figure 1.: Slice through the simulation domain.
</figcaption>

Slicing functionality can be combined with any type of View mode (point cloud, vector fiels, etc.). At the same time, you can 
toggle multiple slicing planes in x/y/z axis (Fig. 2).

<Image
    src="/img/simlai-simulation-studio-slice-2.png"

/>
<figcaption style={{ textAlign: "center", fontSize: "0.7rem", marginBottom: "1rem" }}>
    Figure 2.: Multiple slices can be combined for any type of visualization type (point cloud, vector field, etc.).
</figcaption>
