# Datasets

Users are able to upload their own datasets to Siml.ai Model Engineer. The datasets can be used to train data-driven or 
hybrid models. The datasets can be uploaded in the following formats:
- CSV
- JSON
- Numpy (.npy, .npz)
- HDF5 (.h5, .hdf5)
